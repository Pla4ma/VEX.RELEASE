# Store Assets

Required assets for App Store and Google Play submission.

## Required Files

### App Icon
- `icon.png` - 1024×1024 px
- Used for iOS App Store and Android Google Play
- Format: PNG with transparency

### Adaptive Icon (Android)
- `adaptive-icon.png` - 1024×1024 px
- Background layer for Android adaptive icons

### Splash Screen
- `splash.png` - 1242×2436 px (iPhone X size)
- Used for app launch screen
- Should match app branding

### Favicon (Web)
- `favicon.png` - 32×32 px
- Used for web version

## App Store Screenshots Required

### iPhone
- 6.7" Display (iPhone 14 Pro Max): 1290×2796
- 6.5" Display (iPhone 14 Plus): 1284×2778
- 5.5" Display (iPhone 8 Plus): 1242×2208

### iPad
- 12.9" Display (iPad Pro): 2048×2732
- 11" Display (iPad Pro): 1668×2388

### Android
- Phone: 1080×1920 or higher
- 7" Tablet: 1080×1920
- 10" Tablet: 1200×1920

## Design Guidelines

1. **Icon**: Simple, recognizable, works at small sizes
2. **Screenshots**: Show actual app UI with key features
3. **Splash**: Clean, minimal, branded background
4. **No text in icons** per store guidelines

## Generation

Generate these assets using:
- Figma/Sketch for design
- GIMP/Photoshop for raster editing
- [App Icon Generator](https://appicon.co/) for resizing
